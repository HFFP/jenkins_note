## 源码管理 git

### Repositories
>Repository URL  仓库地址

>Credentials 资格证书
    
    填入仓库地址即可,可以通过 http://username:password@XXX/XXX.git 方式来验证，
    这样可以不用证书，但是在测试的时候，我使用的git私有源，使用这种方式无法clone下来代码
    所以最后还是添加了证书后才能clone下来, 配置证书如下图
    选择Username with password 并填写 Username 和 Passworde 即可
